Blender Add-on

Download and install the companion add-on to send render updates to your iPhone or iPad in real time.

Download add-on (.zip) Read install guide
1
In Blender: Edit → Preferences → Add-ons → Install… then select the downloaded .zip file.
2
Enable Render Monitor in the Add-ons list.
3
Open the Render Monitor panel, click Link Device (Get Code), then enter the code in your iOS app.
4
Start a render (F12 → Render Animation) and watch it appear on your phone.

FAQs

Why is the Image Preview not visible?

Image Preview is a premium feature and is only available for Pro users.

My subscription is not showing up. What should I do?

Simply close and reopen the app. Your subscription status should refresh automatically.

Which Blender versions are supported?

The app currently supports Blender version 3.0 and above.

Which iOS versions are supported?

The app requires iOS 17.0 or later.

I changed my iOS device, and my Pro subscription is missing. How can I restore it?

Tap the Pro button in the top-right corner. When the paywall sheet appears, select Restore Purchases. Make sure you’re signed in with the same Apple ID that you used for the original purchase.

Do I need to keep my computer on for monitoring to work?

Yes. Your computer must remain on and Blender must be running the Render Monitor add-on during rendering for updates to appear in the app.

Can I use my account on multiple devices?

Yes. As long as you sign in with the same account, your subscription and render data will sync across devices.

How often are updates shown in the app?

Status updates and progress sync automatically, and Pro users also receive image previews every few frames.
